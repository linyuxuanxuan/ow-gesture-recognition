# 守望先锋体感游戏源码分享(ow-gesture-recognition)


### 1. 运行步骤  
```
a. git clone https://github.com/linyuxuanxuan/ow-gesture-recognition.git  
b. cd ow-gesture-recognition  
c. pip install -r requirements.txt  
d. python ow.py
```
